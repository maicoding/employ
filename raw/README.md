# Raw

Hier liegen nur public-safe Rohnotizen.

Nicht ablegen:

- geschuetzte Volltexte
- Pressespiegel
- Paywall-Artikel
- interne Dokumente
- personenbezogene Daten

Gute Raw-Notiz:

- URL
- Datum
- eigene kurze Notiz
- warum relevant
- welche Frage offen bleibt
